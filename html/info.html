Securly Classroom

Securly Classroom is a new program OHS is using for the rest of this year, and likely next year. These images and information are from Securly Docs and do not show the OHS admin portal. Note the school may not use all features documented by Securly Classroom.

Location Tracking and Monitoring

Securly Classroom can track student locations and stores screenshots that staff can capture discreetly.

Location Tracking Details

Close-up view of Securly Classroom's geolocation tracking features.

Whitelisting and Blacklisting

Teachers can block websites for individuals or entire classes, with restrictions potentially extending beyond class time. They can also limit access to approved sites only.

IP Address Monitoring

Securly Classroom uses IP addresses to determine location, potentially restricting off-campus device usage or detecting VPN/hotspot bypass attempts.

Real-Time Monitoring

Staff can view screens, capture images, manage tabs, and redirect browsing sessions without student knowledge.
